3. html파일 include 하는 방법

A.html 파일에서 B.html파일의 내용을 보여주는 방법입니다.

3-1. data-html

특정 태그에 data-html속성을 지정하면 속성값에 해당하는 파일의 내용을 보여줍니다.

예제

<div data-html="import.html"></div>

위와같은 태그를 작성하면 렌더링 후 아래처럼 보여집니다.

<div>import.html파일이 이곳에 보여집니다.</div>

3-2. data-replace-html

특정 태그에 data-replace-html속성을 지정하면 속성값에 해당하는 파일의 내용으로 교체합니다.

예제

<p>html교체</p>
<div data-replace-html="import.html"></div>
<p>위 태그 자리에 import.html파일의 내용이 보여집니다.</p>

위와같은 태그를 작성하면 실제 화면은 아래처럼 보여집니다.

<p>html교체</p>
import.html파일이 이곳에 보여집니다.
<p>위 태그 자리에 import.html파일의 내용이 보여집니다.</p>
