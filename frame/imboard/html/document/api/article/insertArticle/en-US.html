3-4-4. $.api.article.insertArticle

#{lan.example}
$.api.article.insertArticle({boardId : "test", subject : "Subject", content : "Content"});
#{lan.parameter}
Variable's name Variable's type Description
boardId Required String Board's id
seq Integer Number of article.
subject String Subject
content String Content
groupId Integer If write a child article, groupId of parent article.
parentSeq Integer If write a child article, seq of parent article.
thumbnailUrl String Article's thumbnail image url
tag String Tag
password String Password for article.
isNotice String(Y or N) Y : is a notice.
N : is a normal article.
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
