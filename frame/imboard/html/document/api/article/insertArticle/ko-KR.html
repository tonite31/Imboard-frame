$.api.article.insertArticle

#{lan.example}
$.api.article.insertArticle({boardId : "test", subject : "Subject", content : "Content"});
#{lan.parameter}
변수명 변수타입 설명
boardId 필수 String 게시판 아이디
seq Integer 게시글 번호
subject String 게시글 제목
content String 게시글 내용
groupId Integer 하위로 게시글을 작성하는경우 상위 게시글의 groupId
parentSeq Integer 하위로 게시글을 작성하는경우 상위 게시글의 seq
thumbnailUrl String 썸네일 주소
tag String 태그
password String 게시글 비밀번호
isNotice String(Y or N) 공지로 지정 여부
#{lan.data}
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
