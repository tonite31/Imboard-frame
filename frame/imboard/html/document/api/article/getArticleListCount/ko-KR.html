3-4-1. $.api.article.getArticleListCount

#{lan.example}
$.api.article.getArticleListCount({boardId : "test", searchData : {subject : "검색할 제목"}});
#{lan.parameter}
변수명 변수타입 설명 검색방법
boardId String 게시판 아이디 =
searchData subject String 게시글 제목 LIKE
writerId String 작성자 아이디 =
likeWriterId String 작성자 아이디 LIKE
writerName String 작성자 이름 =
likeWriterName String 작성자 이름 LIKE
tag String 태그 LIKE
status Integer 게시글 상태 =
readStatus Integer 로그인 사용자의 게시글 읽음 상태 =
writerDisplayId String 작성자 별명 =
likeWriterDisplayId String 작성자 별명 LIKE
registerDate String 게시글 작성날짜 =
existThumbnailUrl Boolean (true or false) true : 썸네일 이미지가 있는 게시글이 검색됩니다.
false : 썸네일 이미지가 없는 게시글이 검색됩니다.		 =
withContent Boolean (true or false) true : 컨텐츠를 포함한 결과를 가져옵니다.
false or empty : 컨텐츠를 포함하지 않은 결과를 가져옵니다.		 =
isNotice String (Y or N) Y : 공지로 지정된 게시글만 검색됩니다.
N : 공지로 지정되지 않은 게시글만 검색됩니다.		 =
orderByGroupId String (DESC or ASC) DESC or empty : 게시글을 groupId 값으로 내림차순 정렬합니다.
ASC : 게시글을 groupId 값으로 오름차순 정렬합니다.		 =
orderBySeq String (DESC or ASC) DESC or empty : 게시글을 seq 값으로 내림차순 정렬합니다.
ASC : 게시글을 seq 값으로 오름차순 정렬합니다.		 =
orderByIsNotice String (DESC or ASC) DESC : 게시글을 isNotice 값으로 내림차순 정렬합니다.
ASC : 게시글을 isNotice 값으로 오름차순 정렬합니다.		 =
orderByRegisterDate String (DESC or ASC) DESC : 게시글을 registerDate 값으로 내림차순 정렬합니다.
ASC : 게시글을 registerDate 값으로 오름차순 정렬합니다.		 =
