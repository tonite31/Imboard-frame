$.api.article.deleteArticle

#{lan.example}
$.api.article.deleteArticle({boardId : "test", seq : 0, isRemove : "Y");
#{lan.parameter}
Variable's name Variable's type Description
boardId Required String Board's id
seq Required Integer Number of article.
isRemove String(Y or N) Y : Delete this article.
N : Set status "-1" to this article.

Variable's name Variable's type Description
code Integer Response code
message Integer Response message
