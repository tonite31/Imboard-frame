3-6-4. $.api.menu.updateMenu

파라미터로 입력된 id에 해당하는 메뉴를 생성한 사용자이거나 관리자이상의 레벨을 가지고 있는 사용자만 정상적으로 수정할 수 있습니다.

예제
$.api.menu.updateMenu({id : "test", name : "테스트용 메뉴 수정", type : 1, priority : 0});
파라미터
변수명 변수타입 설명
id 필수 String 아이디
name String 이름
type Integer 유형
parameter String 파라미터
priority Integer 우선순위
parentMenuId String 상위메뉴 아이디
viewLevel Integer 메뉴보기 권한레벨
데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
