$.api.menu.updateMenu

If a value "creator" of menu for update is not null, then can update by current logined user who is a administrator or id of logined user is equals to a value "creator".

#{lan.example}
$.api.menu.updateMenu({id : "test", name : "테스트용 메뉴 수정", type : 1, priority : 0});
#{lan.parameter}
Variable's name Variable's type Description
id Required String id
name String name
type Integer type
parameter String Url querystring. use this, if you connect menu to board. ex)?boardId=test
priority Integer Priority
parentMenuId String Id of parent menu.
viewLevel Integer User's level of viewing a menu.
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
