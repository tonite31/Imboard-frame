3-6-3. $.api.menu.insertMenu

예제
$.api.menu.insertMenu({id : "test", name : "테스트용 메뉴", type : 0, parameter : "?id=test", priority : 0, viewLevel : 10});
파라미터
변수명 변수타입 설명
id 필수 String 메뉴 아이디
name String 메뉴 이름
type Integer 유형
parameter String 파라미터
priority Integer 우선순위
parentMenuId String 상위메뉴 아이디
viewLevel Integer 메뉴보기 권한레벨
creator String 메뉴를 생성한 사용자 아이디
데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
