3-6-5. $.api.menu.deleteMenu

If a value "creator" of menu for delete is not null, then can delete by current logined user who is a administrator or id of logined user is equals to a value "creator".

#{lan.example}
$.api.menu.deleteMenu({id : "test"});
#{lan.parameter}
Variable's name Variable's type Description
id Required String id
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
