3-5-6. $.api.comment.deleteComment

#{lan.example}
$.api.comment.deleteComment({boardId : "test", articleSeq : 0, seq : 0, isRemove : "Y"});
#{lan.parameter}
Variable's name Variable's type Description
boardId Required String Board's id
articleSeq Required Integer Article's number
seq Required Integer Comment's number
isRemove String(Y or N) Y : Delete this article.
N : Set status "-1" to this article.
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
