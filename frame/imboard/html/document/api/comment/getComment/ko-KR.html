3-5-3. $.api.comment.getComment

예제

게시글을 볼 수 있는 권한이 있어야만 덧글 데이터를 가져올 수 있습니다.

$.api.comment.getComment({boardId : "test", articleSeq : 0, seq : 3});
파라미터
변수명 변수타입 설명 검색방법
boardId 필수 String 게시판 아이디 =
articleSeq 필수 Integer 게시글 번호 =
seq필수 Integer 덧글 번호 =
데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
data commentList boardId String 게시판 아이디
articleSeq Integer 게시글 번호
seq Integer 덧글 번호
content String 내용
registerDate String 작성날짜
good Integer 추천수
bad Integer 비추천수
writerId String 작성자 아이디
writerName String 작성자 이름
writerDisplayId String 작성자 별명
parentWriterId String 상위 덧글 작성자 아이디
parentWriterName String 상위 덧글 작성자 이름
parentWriterDisplayId String 상위 덧글 작성자 별명
writerProfileImgUrl String 작성자 프로필 이미지 주소
parentSeq Integer 상위 덧글 번호
groupId Integer 게시글 그룹 아이디
status Integer 덧글 상태
