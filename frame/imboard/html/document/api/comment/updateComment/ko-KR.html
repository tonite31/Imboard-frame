3-5-5. $.api.comment.updateComment

예제
$.api.comment.updateComment({boardId : "test", articleSeq : 0, seq : 0, content : "덧글내용"});
파라미터
변수명 변수타입 설명
boardId 필수 String 게시판 아이디
articleSeq 필수 Integer 게시글 번호
seq 필수 Integer 덧글 번호
content String 덧글 내용
데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
