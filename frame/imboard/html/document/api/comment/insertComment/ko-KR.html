3-5-4. $.api.comment.insertComment

예제
$.api.comment.insertComment({boardId : "test", articleSeq : 0, content : "덧글내용"});
파라미터
변수명 변수타입 설명
boardId 필수 String 게시판 아이디
articleSeq 필수 Integer 게시글 번호
content String 덧글 내용
groupId Integer 하위로 덧글을 작성하는경우 상위 덧글의 groupId
parentSeq Integer 하위로 덧글을 작성하는경우 상위 덧글의 seq
데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
