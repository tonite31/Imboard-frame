3-7-5. $.api.user.updateUser

예제
$.api.user.updateUser({id : "test", name : "테스트용 사용자 수정", writerProfileImgUrl : "http://localhost:3000/test.jpg"});
파라미터
변수명 변수타입 설명
id 필수 String 사용자 아이디
displayId String 사용자 별명
name String 사용자 이름
writerProfileImgUrl String 프로필 이미지 주소
level Integer 레벨
userStatus String (Y or N) 계정상태 Y : 사용가능, N : 사용불가능
결과데이터
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
