$.api.board.deleteBoard

If a value "creator" of board for delete is not null, then can delete by current logined user who is a administrator or id of logined user is equals to a value "creator".

#{lan.example}
$.api.board.deleteBoard({id : "test"});
#{lan.parameter}
Variable's name Variable's type Description
id Required String Board's id
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
