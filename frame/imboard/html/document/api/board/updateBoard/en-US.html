3-3-4. $.api.board.updateBoard

If a value "creator" of board for update is not null, then can update by current logined user who is a administrator or id of logined user is equals to a value "creator".

#{lan.example}
$.api.board.updateBoard({id : "test", name : "Test board", priority : 0, description : "This is a board for test.", useYn : "Y"});
#{lan.parameter}
Variable's name Variable's type Description
id Required String Board's id
name String Board's name
priority Integer Priority
description String Description
useYn String Use Yn
#{lan.data}
Variable's name Variable's type Description
code Integer Response code
message Integer Response message
