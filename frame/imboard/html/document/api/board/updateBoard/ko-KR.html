$.api.board.updateBoard

만약 수정하려는 게시판의 creator값이 null이 아니라면 현재 로그인한 사용자가 관리자이거나 아이디가 creator값과 일치해야만 수정이 가능합니다.

#{lan.example}
$.api.board.updateBoard({id : "test", name : "테스트용 게시판", priority : 0, description : "테스트용 게시판입니다.", useYn : "Y"});
#{lan.parameter}
변수명 변수타입 설명
id 필수 String 게시판 아이디
name String 게시판 이름
priority Integer 우선순위
description String 설명
useYn String 사용여부
#{lan.data}
변수명 변수타입 설명
code Integer 통신결과코드
message Integer 통신결과메시지
