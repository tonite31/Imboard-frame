3-3. scrollFollower

{{#localize}} @{ko-KR} 스크롤이 되었을때 스크롤을 따라서 움직이는 엘리먼트를 만들어주는 컴포넌트입니다. @{default} This is a component that is moved by scroll. {{/localize}}

#{lan.code}
<ul data-component="scrollFollower" id="scrollFollower">
	<li>Menu1</li>
	<li>Menu2</li>
	<li>Menu3</li>
</ul>

<script>
$(document).ready(function()
{
	$("#scrollFollower").compile({direction : "vertical", top: 20});
});
</script>
Options
direction"vertical" or "horizontal"
{{#localize}}@{ko-KR}스크롤의 방향을 결정합니다. 기본값은 "vertical" 입니다.@{default}Direction of scroll. default value is a "vertical"{{/localize}}
topnumber
{{#localize}}@{ko-KR}top으로부터 떨어질 px값입니다. direction이 "vertical" 일때만 동작합니다. 기본값은 10입니다.@{default}If direction value is a "vertical", then apply this value to style of top. default value is a 10.{{/localize}}
leftnumber
{{#localize}}@{ko-KR}left로부터 떨어질 px값입니다. direction이 "horizontal" 일때만 동작합니다. 기본값은 10입니다.@{default}If direction value is a "horizontal", then apply this value to style of left. default value is a 10.{{/localize}}
