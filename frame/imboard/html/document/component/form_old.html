API
{{#localize}} @{ko-KR}
$(form).compile(callback)form 엘리먼트의 submit 버튼이 클릭됐을때 callback이 호출되며 getData함수의 결과값을 인자로 넘겨줍니다.
$(form).getData()form 엘리먼트 하위의 name속성 또는 data-name속성을 가지고 있는 엘리먼트의 특정값을 JSON형태로 return합니다.
$(form).setData(data)JSON형태의 데이터를 인자로 받아들이며 key값과 일치하는 name속성 또는 data-name속성을 가진 엘리먼트에 value값을 세팅해줍니다.
@{default}
$(form).compile(callback)Call a callback function with parameter that is returned from "getData" function when form submit.
$(form).getData()Return a data of JSON type that has a attribute "name" or "data-name" in form.
$(form).setData(data)Parameter "data" is a object of JSON type. set a data to tag that has a attribute "name" or "data-name" and equals value to key of data.
