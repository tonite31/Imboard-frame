1. data-bind 모듈

1-1. 개요

data-bind 모듈은 HTML 코드만으로 서버 혹은 데이터베이스로부터 데이터를 불러와서 화면에 조합하는 기능입니다. 사용법은 다음과 같습니다.

<div data-bind="모듈명" data-param='{"key" : "value"}' data-template-id="템플릿 아이디"></div>

위와 같은 HTML태그를 작성하면 화면이 렌더링될때 모듈명에 해당하는 모듈을 실행하고 결과 데이터를 템플릿으로 넘겨줍니다. data-param 속성값은 json유형으로 설정해야합니다.

템플릿은 Handlebars 문법을 사용하여 작성합니다.
