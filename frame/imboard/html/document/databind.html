2. 데이터조합

미리 정의된 치환변수를 사용하거나 HTML 엘리먼트에 data-bind속성을 지정하면 간편하게 서버로부터 데이터를 불러와서 화면에 조합할 수 있습니다.

2-1. 치환변수

치환변수는 #{} 형태로 사용하며 서버의 특정 변수로 치환됩니다. 치환변수를 문자 그대로 사용하려면 #을 하나 더 붙인 ###{path.key} 형태로 입력합니다.

##{path.root}프레임의 루트경로입니다. 프레임의 이름이 sample이라면 치환되는 값은 /content/frame/sample 입니다.
##{path.module}프레임의 공통 모듈경로입니다. 치환되는 값은 /content/module 입니다.
##{path.lib}프레임의 공통 라이브러리경로입니다. 치환되는 값은 /content/lib 입니다.
##{query.xxx}xxx키에 해당하는 querystring값으로 치환됩니다. ##{query}만 지정할경우 ?를 포함한 querystring 전체가 치환됩니다.
##{user.xxx}로그인한 사용자의 정보중 xxx키에 해당하는 값으로 치환됩니다.
##{locale}접속한 클라이언트의 언어 코드 값으로 치환됩니다. 만약 querystring에 locale값이 있을경우 그 값으로 치환됩니다. ex)ko-KR, en-US
##{lan.xxx}querystring에 locale값이 있다면 해당 값으로, 그렇지 않으면 접속한 클라이언트의 언어에 따라 localize.json에 입력된 xxx키에 해당하는 값으로 치환됩니다. 국제화방법은 6. 국제화를 참조하십시오.

2. Data composition

Can composite a data to screen by a replace variables or a attribute "data-bind" to specific html tag.

2-1. Replace variables

Replace variable is used by grammar of "#{}", and changed to specific variable of server. if want to use a original word "##{path.key}", then use a "###{path.key}".

##{path.root}Root path of a frame. if frame's name is the "sample", then replace value is "/content/frame/sample".
##{path.module}The path of common module of a frame. replace value is "/content/module".
##{path.lib}The path of common library of a frame. replace value is "/content/lib"
##{query.xxx}Is replaced to value of specific querystring that equals a key "xxx". if use a "##{query}", then replaced by all querystring.
##{user.xxx}Is replaced to value of specific signined user infomation that equals a key "xxx".
##{locale}Is replaced to value of language code of connected client. if querystring has a key "locale", then replaced by it. ex)ko-KR, en-US
##{lan.xxx}Is replaced to value of language of connected client. if querystring has a key "locale", then replaced by use it. Usage refer to 6. Localize

2-2. data-bind

{{#localize}} @{ko-KR}

2-2-1. 개요

data-bind 모듈은 HTML 태그에 data-bind, data-param, data-template-id 속성을 지정해서 서버의 모듈을 통해 얻어온 데이터를 화면에 조합할 때 사용합니다.

data-bind 속성값에 해당하는 모듈을 실행하고, 그 결과 데이터를 Handlebars 문법을 통해 화면에 조합합니다.

ex) <div data-bind="모듈명" data-param='{"key" : "value"}' data-template-id="템플릿 아이디"></div>

@{default}

2-2-1. Overview

data-bind module is used to composite a data from the server module by set a attribute "data-bind", "data-param", "data-template-id" to specific tag.

Composite a data from the server module by Handlebars.

ex) <div data-bind="module's name" data-param='{"key" : "value"}' data-template-id="template's id"></div>
