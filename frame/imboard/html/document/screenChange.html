2. 화면을 전환하는 방법

2-1. 전체화면전환

URL에 지정된 html파일을 화면에 보여줍니다.

http://localhost:3000//content/frame/프레임명/html/ 디렉토리의 index.html 파일을 화면에 보여줍니다.
http://localhost:3000/test/test.html/content/frame/프레임명/html/test/ 디렉토리의 test.html 파일을 화면에 보여줍니다.
http://localhost:3000/signin로그인화면으로 이동
http://localhost:3000/signout.do로그아웃
http://localhost:3000/settings관리자 대쉬보드 화면으로 이동

2-2. 부분화면전환

URL의 특정 파라미터에 지정된 html파일로 화면 일부를 전환합니다. 이 기능을 사용하기 위해서는 화면조각영역으로 사용할 태그에 data-fragment속성을 지정해야 합니다. 속성값은 다른 URL파라미터와 중복되지않는 값으로 지정하면 됩니다.

만약 특정 태그에 data-fragment="body" 속성이 지정되어있을때 URL파라미터에 body=test가 지정되면 각 프레임의 html디렉토리 하위의 test.html의 내용이 태그에 보여지고 js디렉토리 하위의 test.js파일과 css디렉토리 하위의 test.css파일이 자동으로 import됩니다.

아래 예제는 div태그에 data-fragment="body" 속성을 지정하고 각 버튼에 body파라미터를 변경하도록 href속성을 지정하였습니다. .html확장자는 생략할 수 있습니다.
