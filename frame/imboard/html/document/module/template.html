template

예제
데이터

template모듈은 특별한 데이터가 필요 없이 치환변수만을 이용해서 템플릿을 작성할때 주로 사용합니다.

코드
<script id="templateTemplate" type="text/x-handlebars-template">
{{#if user}}
<p>{{user.name}}님 환영합니다.</p>
{{else}}
<p>로그인하지 않은 사용자입니다.</p>
{{/if}}
</script>

<div data-bind="userList" data-template-id="userListTemplate"></div>
