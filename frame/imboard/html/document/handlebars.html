2-3. Handlebars Helper

{{#localize}} @{ko-KR} data-bind 모듈로 가져온 데이터는 Handlebars 문법으로 화면에 조합됩니다. 여기서는 추가적으로 구현된 Helper 문법을 소개하며 Handlebars의 기본적인 문법은 Handlebars 홈페이지를 참고하십시오. @{default} The data from "data-bind" module is composited to screen by Handlebars grammar. Following description is additional helper of handlebars. Please refer of basic grammar to Handlebars {{/localize}}

equals {{#localize}} @{ko-KR} 첫번째 파라미터와 두번째 파라미터가 일치하면 true, 일치하지 않으면 false @{default} if equals 1st paramter to 2nd paramter, then true else false. {{/localize}}
{{#equals param1 param2}} ... {{else}} ... {{/equals}}
notequals {{#localize}} @{ko-KR} 첫번째 파라미터와 두번째 파라미터가 일치하지 않으면 true, 일치하면 false @{default} if not equals 1st paramter to 2nd paramter, then true else false. {{/localize}}
{{#notequals param1 param2}} ... {{else}} ... {{/equals}}
empty {{#localize}} @{ko-KR} 첫번째 파라미터가 빈값이면 true, 그렇지 않으면 false @{default} if 1st parameter is empty, then true else false. {{/localize}}
{{#empty param}} ... {{else}} ... {{/equals}}
notempty {{#localize}} @{ko-KR} 첫번째 파라미터가 빈값이 아니면 true, 그렇지 않으면 false @{default} if 1st parameter is not empty, then true else false. {{/localize}}
{{#notempty param}} ... {{else}} ... {{/equals}}
tojson {{#localize}} @{ko-KR} 첫번째 파라미터가 JSON형태의 object이고 두번째 파라미터가 object의 key일때 value를 리턴합니다. @{default} if 1st paramter is a object of JSON type and 2nd paramter is a key of 1st parameter, then return value of key. {{/localize}}
{{tojson param1 param2}}
math {{#localize}} @{ko-KR} 첫번째 파라미터와 세번째 파라미터를 두번째 파라미터로 연산한 결과를 리턴합니다. @{default} Return a value that is calculated by 2nd parameter with 1st parameter and 3rd parameter. {{/localize}}
{{math param1 "(+, -, *, /, %)" param3}}
sign {{#localize}} @{ko-KR} 첫번째 파라미터와 세번째 파라미터를 두번째 파라미터로 비교한 결과를 리턴합니다. @{default} Return a result that is compared by 2nd parameter with 1st parameter and 3rd parameter. {{/localize}}
{{sign param1 "(<, lt, <=, lt=, >, gt, >=, gt=, ==, ===, !=, !==)" param3}}
shortDate {{#localize}} @{ko-KR} 첫번째 파라미터가 yyyy-MM-dd hh:mm:ss형식일때 오늘이면 hh:mm:ss형태로 오늘이 아니면 yyyy-MM-dd 형태로 리턴합니다. @{default} if 1st parameter is a type of date "yyyy-MM-dd hh:mm:ss" and is a today, then return value of date type "hh:mm:ss" else return value of date type "yyyy-MM-dd". {{/localize}}
{{shortDate param1}}
escape {{#localize}} @{ko-KR} 첫번째 파라미터를 escape처리해서 리턴합니다. @{default} Return a value that is escaped to 1st parameter. {{/localize}}
{{escape param1}}
