3-1. 개요

imboard-api는 서버로 요청을 보내고 응답을 받는 함수들과 각종 데이터를 제어할 수 있는 객체로 구성된 라이브러리입니다. 반드시 /content/module/imboard/js/imboard-api.js 자바스크립트 파일을 index.html파일에 import해야 사용할 수 있습니다.

3-1. Overview

imboard-api is a library that has functions that request to server and object that control to client data. for using this library, must be imported /content/module/imboard/js/imboard-api.js to index.html.
