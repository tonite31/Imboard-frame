1. 화면 전환 및 조각 구성방법

화면전환은 전체페이지와 조각페이지 단위로 전환할 수 있습니다. 전체페이지는 완전한 HTML문서로 제작되어야 하며 페이지 조각은 일부의 엘리먼트로만 제작할 수 있습니다. 페이지 조각은 다른 페이지 조각을 포함하거나 전체 페이지의 일부로 포함될 수 있습니다.

1-1. 전체페이지

전체페이지는 URL에 입력된 디렉토리 구조에 해당하는 페이지가 보여집니다.

http://localhost:3000/##{path.root}/index.html
http://localhost:3000/test/test.html##{path.root}/test/test.html

1-2. 조각페이지

조각페이지를 보여주기 위해서는 HTML문서의 특정 태그에 data-fragment속성을 지정해야합니다. URL파라미터 중 data-fragment의 속성값을 키로 가지는 값에 해당하는 html파일을 해당 엘리먼트 하위로 보여줍니다.

보여지는 페이지 조각의 경로와 일치하는 css, js파일도 함께 import됩니다.

1-3. 내부페이지

	
조각페이지와 성격이 비슷하지만 내부페이지는 조각페이지처럼 파라미터에 의해 페이지가 변하지 않는 고정페이지 입니다.

	
내부페이지는 특정 엘리먼트에 data-html속성을 지정하면 사용할 수 있으며 data-html의 속성값으로는 html의 파일 경로를 지정합니다. 파일경로는 프레임의 html폴더를 기준으로 상대경로입니다.
