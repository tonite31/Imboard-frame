    사전준비

    1. 웹서버

    임보드를 설치할 호스팅 서버 또는 개인 서버가 필요합니다.

    2. nodejs & MySQL

    임보드는 nodejs기반 플랫폼이며 데이터베이스는 MySQL을 사용합니다. 최신버전의 nodejs와 MySQL을 다운로드하여 설치해주시기 바랍니다.



    다운로드 및 설치

    1. 임보드 다운로드

    홈페이지에서 임보드를 다운로드 하거나 커맨드라인에서 npm install imboard를 입력해서 설치합니다.

    2. 필요한 nodejs 모듈 설치

    임보드의 루트 디렉토리에서 npm install 명령어를 수행합니다.

    3. 데이터베이스 및 사용자 생성

    설치한 MySQL에 접속한 뒤, 사용자와 데이터베이스를 생성합니다.

    4. 데이터베이스 접속정보 설정

    임보드 루트 디렉토리에서 node config.js 명령어를 수행합니다. 4번 jdbc메뉴를 선택한 후 host, username, password, database 정보를 차례로 입력합니다.

    5. 초기데이터입력

    임보드의 루트 디렉토리에서 node install.js 명령어를 수행합니다. 4번에서 설정한 데이터베이스로 초기데이터가 입력되고 임보드 설치작업이 끝납니다.

    Advance preparation

    1. Web server
    You need a server for install imboard.
    2. nodejs & MySQL
    Imboard use nodejs and MySQL database. Install recent version nodejs and MySQL.

    Download and setup.

    1. Download Imboard
    Download Imboard from homepage or command npm install imboard on CLI
    2. Install dependency nodejs module.
    Command npm install on CLI.
    3. Create database and user on MySQL
    Create database and user after connect to installed MySQL.
    4. Configuration
    Command node config.js on CLI. select a number 4. enter the host, username, password, database name.
    5. Insert initial datas.
    Command node install.js on CLI.
    {{/localize}}


    Run application

    {{#localize}} {{"ko-KR"}}
    1. frame 설정
    임보드 루트 디렉토리에서 node config.js 명령어를 수행합니다. 1번 frame메뉴를 선택한 후 프레임 이름을 입력합니다.
    프레임 이름은 /content/frame/ 디렉토리 하위의 폴더 이름입니다. 기본값은 blog입니다.
    2. server 정보 설정
    1번에 이어서 2번 server메뉴를 선택한 후 port와 host정보를 입력합니다.
    host는 임보드 어플리케이션으로 접속하는 웹 URL입니다. ex)port=3000, host=http://localhost:3000
    3. 임보드 어플리케이션 실행
    임보드의 루트 디렉토리에서 node app.js 명령어를 수행합니다.
    4. 웹페이지 접속
    웹브라우저를 열어서 주소입력부분에 http://localhost:3000/ 을 입력합니다.
    {{"default"}}
    1. Set frame
    Command node config.js on CLI. select a number 1. enter a frame name.
    Frame name is directory name in /content/frame/ directory. default value is blog.
    2. Set server information
    Select a number 2. enter a port and host.
    host is web url for application. ex)port=3000, host=http://localhost:3000
    Quit a configuration by enter a letter "q"
    3. Run application
    Command node app.js on CLI
    4. Connect webpage
    Open your web browser. acess in http://localhost:3000/
