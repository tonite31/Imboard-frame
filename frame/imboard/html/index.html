    사전준비

    1. 웹서버
    임보드를 설치할 호스팅 서버 또는 개인 서버가 필요합니다.
    2. nodejs & MySQL
    임보드는 nodejs기반 플랫폼이며 데이터베이스는 MySQL을 사용합니다. 최신버전의 nodejs와 MySQL을 설치해주시기 바랍니다.
    3. 데이터베이스 및 사용자 생성
    설치한 MySQL에 접속한 뒤, 사용자와 데이터베이스를 생성합니다.

    설치

    아래의 설치과정은 모두 커맨드라인에서 진행됩니다. cmd, terminal, shell등을 사용하여 임보드가 설치된 디렉토리로 이동해서 명령어를 수행합니다.
    1. nodejs 모듈 설치
    npm install
    2. 데이터베이스 접속정보 설정
    node config.js
    jdbc메뉴를 선택한 후 host, username, password, database 정보를 차례로 입력합니다.
    3. 초기데이터입력
    node install.js
    설정한 데이터베이스로 초기데이터가 입력되고 임보드 설치작업이 끝납니다.
    @{default}

    Advance preparation

    1. Web server
    You need a server for install imboard.
    2. nodejs & MySQL
    Imboard use nodejs and MySQL database. Install recent version nodejs and MySQL.
    3. Create database and user on MySQL
    Create database and user after connect to installed MySQL.

    Installation step is progressed on command-line. execute command on the directory that was installed imboard using cmd or terminal or shell.
    1. Install dependency nodejs module.
    npm install
    2. Configuration
    node config.js
    select a number 4. enter the host, username, password, database name.
    3. Insert initial datas.

    #{lan.runapplication}

    {{#localize}} @{ko-KR} 
    node config.js
    어플리케이션을 실행하기 위한 설정값을 입력합니다. 변경사항이 없다면 최초에 한번만 입력하면 됩니다.
    1. frame 설정
    frame메뉴를 선택한 후 프레임 이름을 입력합니다.
    프레임 이름은 /content/frame/ 디렉토리 하위의 폴더 이름입니다. 기본값은 blog입니다.
    2. host, port 설정
    1번에 이어서 2번 server메뉴를 선택한 후 port와 host정보를 입력합니다.
    host는 임보드 어플리케이션으로 접속하는 웹 URL입니다. ex)port=3000, host=http://localhost:3000
    3. 어플리케이션 실행
    node app.js
    4. 웹페이지 접속
    http://localhost:3000/
    @{default} 
    node config.js
    Enter a value for a application. if you has not change, enter a value only once in a first time.
    1. Set a frame's name
    Select a number 1. enter a frame's name.
    Frame name is directory name in /content/frame/ directory. default value is blog.
    2. Set a host and a port.
    Select a number 2. enter a port and host.
    host is web url for application. ex)port=3000, host=http://localhost:3000
    3. Run application
    node app.js
    4. Connect webpage
    http://localhost:3000/
